# 스프링 MVC 1편

---
### 07.25 수업을 마무리하며

 김영한 님의 스프링 MVC 강의 중 1편에 대한 강의를 모두 마무리 하였다. 개인적으로 스프링에 대해 좀 더 자세하게 알 수 있는 시간이었다.

 강의에서 제일 좋았던 부분은 스프링 구조를 코드를 통해서 하나씩 만들어보며 알 수 있었던 점이었다. 항상 스프링을 사용하면서도 어떻게 
동작하는지에 대한 궁금증이 늘 남아 있었고, 구조를 한번 보고 싶어도 코드도 많고 어디를 어떻게 따라가야할 지 감도 잡히지 않아서 개발 할 때마다
항상 눈대중으로 이렇겠지~ 하면서 사용하던 부분이 많았다. 이번 강의를 들으면서 DispatcherServlet의 역할과 FrontController 패턴에
대해서도 알 수 있었고, 스프링이 FrontController 패턴으로 만들어졌단 것도 알 수 있었다.

 이 뿐만 아니라 스프링에서 사용하는 기본 기능들도 다시 한 번 공부하면서 정리할 수 있는 시간이었다. HttpEntity의 경우 최근에 사용하면서 
알게 되었는데, 여기서도 똑같이 다루어줘서 해당 내용에 대해 공부할 수 있게 되었다.

 개인적으로는 바로 2편을 들어가기 보단 이번주는 1편에 대한 복습 위주로 한번 더 진행 해보려 한다. 물론 시간이 어느정도 있어야 하겠지만...
그래도 한번 내 나름대로 정리 한 후 2편으로 넘어가는게 조금 더 확실한 정리가 될 것 같다. (그리고 다음주 포텐데이를 생각하면... 흐름도 끊길 거 같고..)

무튼 매일 조금씩 짬내서 공부하던 MVC 1편을 끝내서 후련한 마음이 크다.
